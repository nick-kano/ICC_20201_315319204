Para compilar el programa siga los siguientes pasos:
1) Abra la terminal.
2) Utilizando el comando cd cambie la dirreccion a la carpeta practica04/src (Ej si la carpeta estuviera en la carpeta de descargas el comando se verÃ­a: cd Downloads/practica04/src).
3) Use el comando "javac Main.java" para compilar el archivo Main.
4) USe el comando "java Main > Main.html" para crear el archivo de internet
5) Use el comando "firefox -new-window Main.html" para abrir el archivo en firefox. (En caso de no tener firefox puede abrir el archivo manualmente).
