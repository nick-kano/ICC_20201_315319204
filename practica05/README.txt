Para compilar el programa siga los siguientes pasos:
1) Abra la terminal.
2) Utilizando el comando cd cambie la dirreccion a la carpeta practica05/src (Ej si la carpeta estuviera en la carpeta de descargas el comando se vería: cd Downloads/practica05/src).
3) Use el comando "javac Control.java" para compilar el archivo Control.
4) Use el comanfo "java Control" y el archivo se ejecutará en la consola.
