Para compilar el archivo siga los siguientes pasos
1) abra la terminal
2) con el comando cd cambie de carpeta al archivo donde tenga esta carpeta, por ejemplo si esta en descargas el comando se veria cd /Downloads/practica09
3) compile el archivo usando el comando javac WriteAndRead.javac
4) el programa leera los puntos del archivo coordenadas.txt y los convertira a cordenadas polares, si quiere agregar mas puntos o quitar algunos puede modificar el archivo coordenadas.txt
5) una vez que quiera convertir los puntos utilice el comando java WriteAndRead y este creara un archivo txt llamado polares donde estara las conversiones a coordenadas polares del archivo cordenadas.txt
